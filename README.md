# Daily News Updater (DNU)

这是一个自动化的新闻聚合与总结工具。它定期从指定的 RSS 源获取新闻，使用大语言模型（LLM）生成摘要，并将最终的简报同步到飞书多维表格中。

## ✨ 功能特点

- **RSS 订阅**：支持配置多个 RSS 源（如塔斯社、路透社等）。
- **智能总结**：
  - **Map 阶段**：逐条总结新闻内容，提取核心事件。
  - **Reduce 阶段**：汇总所有摘要，生成按主题分类的深度简报（政治、经济、地缘局势等）。
- **飞书同步**：自动将生成的简报写入飞书多维表格。
- **自动化运行**：支持 GitHub Actions 每日定时执行。
